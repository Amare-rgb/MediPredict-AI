# 🏥 MediPredict AI

## AI-Powered 30-Day Hospital Readmission Prediction System

### 📌 Project Overview

MediPredict AI is a sophisticated machine learning solution that predicts the likelihood of hospital readmission within 30 days of discharge. Designed for the HABTech Solutions AI Healthcare Analytics Challenge 2026, this system combines state-of-the-art XGBoost modeling with transparent SHAP explanations to support clinical decision-making.

### 🎯 Key Features

- **Real-time Risk Prediction**: Instant assessment of readmission risk with probability scores
- **SHAP Explainability**: Transparent, patient-specific explanations of risk factors
- **Clinical Recommendations**: Actionable insights based on risk level and patient profile
- **Interactive Dashboard**: Professional Streamlit interface with real-time analytics
- **Model Performance**: 87% recall, optimized to minimize false negatives

### 🛠 Technology Stack

| Component | Technology |
|-----------|------------|
| ML Framework | XGBoost |
| Dashboard | Streamlit |
| Explainability | SHAP |
| Visualization | Plotly, Matplotlib |
| Data Processing | Pandas, NumPy |
